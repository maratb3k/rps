# 🎮 RPS — Rock Paper Scissors (iOS)

Классическое iOS-приложение игры **«Камень, Ножницы, Бумага»** против компьютера.  
Простая реализация с поддержкой разных режимов и визуальной обратной связью.

---

## ✨ Возможности

### 🎲 Режимы игры

**Автоматический режим**
- Выбор игрока генерируется случайно  
- Одно нажатие — один раунд  

**Ручной режим**
- Игрок выбирает:
  - Камень 🪨  
  - Ножницы ✂️  
  - Бумагу 📄  

---

### 🔁 Игровая логика

- Серия из **3 раундов**
- Итоговый результат после завершения серии

---

### 📊 Статистика

- Отображение в реальном времени:
  - Победы 🟢  
  - Поражения 🔴  
  - Ничьи ⚪️  

---

### 🎨 UI/UX

- Подсветка выбранного элемента  
- Эффект масштабирования  

---

## 🛠 Технологии

- Swift  
- UIKit  
- Storyboard (Interface Builder)  
- MVC
